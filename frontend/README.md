# Phase 4B Frontend

React + Vite enterprise UI for the SAP P2P intelligence platform.

## Stack

- React 18
- TypeScript
- React Router
- TanStack Query
- Recharts

## Run locally

```powershell
cd frontend
npm install
npm run dev
```

## Build

```powershell
cd frontend
npm run build
```

## Environment

Copy `.env.example` to `.env` if you want to override the API base URL.
